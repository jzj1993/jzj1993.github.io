太白山北南穿越全程图解攻略(3)——出游准备

发表于 2015-06-18 分类于 学习思考 阅读次数： Valine： 本文字数： 3.2k 阅读时长 ≈ 3 分钟

这部分结合实际体验，介绍出游前的准备工作。

关于高原反应

山上气压低缺氧，容易高原反应，也是我一开始比较担心的问题。

高原反应症状

  1. 爬山的时候容易喘气和腿酸，有点像跑了一千米，但是休息一下症状很快缓解消失。
  2. 轻微头疼，大脑不清醒，有点像没睡好觉的感觉；此时爬山需集中注意力，以免发生意外。
  3. 高山区域待的时间稍久一点，高原反应就会减轻。
  4. 根据个人体验和网上攻略，晚上睡觉时头疼可能会加剧，有点像感冒发烧了头疼。
  5. 住在高原地区（海拔500m即为高原地区，不一定有3000m）的孩子可能不会高原反应，或者很轻微。
  6. 通常年龄小更容易适应新环境（低气压），高原反应也会相对轻微一些。
  7. 如果海拔上升慢，高原反应症状会减轻。

建议

  • 避免剧烈运动（刚下车不一定会有反应，但爬山可能就有反应了，所以动作尽量轻缓一些）。
  • 如有轻微头晕，可以坐下深呼吸；可吃生姜、阿莫西林、晕车药缓解（不过吃药容易犯困）。
  • 头疼属于正常反应，需要慢慢适应。如果感觉很严重，确实受不了，安全起见可考虑原路返回。
  • 正常体质的年轻人，高原反应除了有些身体不适，基本不会出现什么问题，所以并不可怕。

食物/水/住宿地点

北南穿越全程，我大概喝了六瓶水。喘气（高原反应）、说话、出汗都很消耗水分，当时晚上热流汗多，吃很干/味道较重的食物也得补充水分。以下地点主要指从北线天圆地方上去之后的线路。

  • 可以买水、买吃的、有人安排住宿（提供床位被褥）：太白山自然保护区售票处、小文公庙、大文公庙、大爷海、药王殿、南天门（我们经过的时候没人）、铁甲树。

  • 在其他一些地方有房子，但是没人安排住宿，可以自己准备大衣，生火住房子里，就不具体介绍了。

  • 如果自带帐篷，随便找个平地住宿就好了，在有些营地扎帐要收费。

  • 流水一般比较干净，灌水的时候动作缓一点免得底下的砂石飘起来。太白第一泉、三合宫瀑布的水都很干净，一般可以直接喝。

  • 不流动的水源（建议不要直接喝，如果自带气罐可煮水，但要考虑到山上低气压水的沸点太低）：大爷海、二爷海、三爷海、玉皇池，其中大爷海的水还能看到小虫子。

摄影器材、数码设备

  • **单反：**我用的佳能650D

  • **镜头：**大部分使用适马10-20/F3.5超广角镜头，少数照片用EFS 18-135 IS。

  • **附件：**渐变滤镜、快门线、内存卡、备用电池

  • 可以带上小型三脚架、单脚架（还能当登山杖用），不建议带大型三脚架、体积大又很沉的长焦镜头等器材，除非身体素质很好。

  • 电池/移动电源/充电线

  • 耳机/电子书/电影等（打发多余时间备用）

食物、饮料参考

注意

  • 独立小包装，方便
  • 重量轻一点的，省力
  • 避免充气包装食品带上顶峰，在气压低的高山可能会爆裂
  • 食物一定要带足，特别是南线下山路很长

高能食品

  • 高能量食物主要作为后备、预防万一
  • 士力架：吃一个顶一顿饭，但是真心太甜
  • 压缩饼干：据说很难吃

参考食品

  • 主食：泡面/自热米饭/面包
    1. 晚上在山顶吃点热的比较好，可买热水泡面
    2. 自热米饭很方便，不过比较沉，西电新校区综合楼多又好等超市有售
  • 补充能量：巧克力/糖果/牛肉粒等（强烈推荐糖果、牛肉粒，饿了随时吃几颗非常方便）
  • 补充盐分：火腿肠/鸡蛋/牛肉粒/豆干/辣条/盐津梅肉/榨菜/干吃面等
  • 补充维生素：维C饮料/果汁等
  • 调味/开胃：辣条等

保健食品

  • 口香糖/木糖醇：防晕车，提神

  • 红牛：提神

  • 生姜：开胃，防晕车，肚子痛，提神，治感冒

饮用水

  • 足够的水（北南穿越建议带2~3瓶水，不够在山上再买）
  • 维C饮料、果汁（可以补充维生素）
  • 脉动（可以补充维生素、无机盐等）
  • 杯子（山上买热水灌杯子里更省钱，部分水源可灌水喝）

着装、户外相关

  1. 厚一点的外套/冲锋衣/羽绒服

    • 我们上山是六月份，山上不冷（没有积雪，大爷海没结冰，晚上盖被子还有点流汗）。作为抗冻的南方人，我就只穿了一条运动裤、长袖，然后带了一件秋天穿的外套，早上在顶峰微冷但没啥问题。
    • 海拔上升100m，气温下降0.6℃。高海拔区温度较低，白天爬山尤其是有太阳不会很冷，但晚上冷，必要时可租大衣。
    • 不要在山下就穿厚衣服，爬山时会热死的。
    • 上山前穿个长袖很方便，热的时候卷起衣袖，有点冷刚好可以放下衣袖。
    • 可考虑带两件厚度适中的衣服，可避免一件厚衣服穿上热不穿又冷的尴尬。
    • 在高海拔区域即使爬山有点流汗，但又有冷风吹着，也不要轻易脱掉衣服，以免感冒。

  2. 运动裤

    • 不建议穿牛仔裤，版型不利于行进，湿了很难打理且变重

  3. 手套、登山杖

    • 手套在爬石头路、比较陡和滑的下坡时，能用上
    • 登山杖看个人需要

  4. 登山鞋 / 好点的运动鞋（防滑，舒适）

  5. 几双厚袜子

    • 几双袜子方便换，脚湿了会很不舒服，厚一点免得冷而且穿着舒服。
    • 袜子千万不敢破个洞，必定磨脚起水泡。

  6. 建议女生别穿高跟鞋/裙子（貌似有女生穿着高跟鞋上山的，真是佩服）

  7. 雨衣、雨伞

    • 山上风大难走时，不便打伞
    • 雨伞可防雨、防晒，山下可用，山上好走的路线应急用一下也行。

  8. 手电

    • 晚上和清晨有用
    • 不少手机有手电功能

  9. 塑料袋

    • 晕车可用
    • 下雨可包裹数码设备防水
    • 可装垃圾
    • 装其他东西

  10. 帽子、墨镜

*   帽子可防晒
*   在山上积雪较多的季节带上墨镜以免反光刺眼，六月份上去积雪不多
  1. 帐篷
*   帐篷比较沉，需要较好的体力
  1. 气罐和炊具
*   可以煮水、煮面，可以在户外用品店、网上买
  1. 手机+户外助手软件/手持GPS
*   如担心迷路，可预先安装户外助手软件，下载谷歌离线卫星地图，用GPS导航
*   卫星地图对于非景区的野外很有用
*   太白山北南穿越路标很多不难走，基本不需要

药品

  • 晕车药（晕车，高原反应）
  • 创可贴（意外划伤）
  • 阿莫西林（缓解高原反应的头晕恶心）
  • 风油精（防晕车，提神，止痒）
  • 云南白药
  • 藿香正气丸

其他

  • 钱（北南两日穿越建议带500）

  • 身份证、学生证（买车票、保险、打折优惠等）

  • 银行卡（刚进景区时可能有取款机）

  • 卫生纸/餐巾纸

  • 别忘了提前一天给移动电源、手机、单反等充满电

  • 前一天晚上一定要休息好

  • 居住在平原地区、爬山经验很少、平时运动少且体能不佳的，建议别去或走简单路线（例如北线原返），北南穿越强度较大，加上高原反应，可能受不了

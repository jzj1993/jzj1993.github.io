MAC安装配置Tomcat

发表于 2016-11-25 分类于 开发技术 阅读次数： Valine： 本文字数： 745 阅读时长 ≈ 1 分钟

从Tomcat官网下载Binary Distributions-Core下的Tomcat.tar.gz，并解压保存到/Library/Tomcat目录。

传送门：
官网 http://tomcat.apache.org/
Tomcat 8.5.8的地址 http://apache.fayea.com/tomcat/tomcat-8/v8.5.8/bin/apache-tomcat-8.5.8.tar.gz

启动

  1. /Library/Tomcat/bin/startup.sh

浏览器输入 http://localhost:8080 进行验证

停止

  1. /Library/Tomcat/bin/shutdown.sh

创建脚本方便使用

  1. sudo vim /usr/local/bin/tomcat

输入以下内容

  1. #!/bin/bash
  2. case $1 in
  3. start)
  4. sudo sh /Library/Tomcat/bin/startup.sh
  5. ;;
  6. stop)
  7. sudo sh /Library/Tomcat/bin/shutdown.sh
  8. ;;
  9. restart)
  10. sudo sh /Library/Tomcat/bin/shutdown.sh
  11. sudo sh /Library/Tomcat/bin/startup.sh
  12. ;;
  13. *)
  14. echo "Usage: startstoprestart"
  15. ;;
  16. esac
  17. exit 0

添加可执行权限

  1. sudo chmod +x /usr/local/bin/tomcat

之后在任意目录下，都可以通过终端启动/停止Tomcat。

  1. # 启动
  2. tomcat start
  3. # 停止
  4. tomcat stop
  5. # 重启
  6. tomcat restart

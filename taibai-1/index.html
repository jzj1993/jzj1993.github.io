太白山北南穿越全程图解攻略(1)——路线策划

发表于 2015-06-18 分类于 学习思考 阅读次数： Valine： 本文字数： 1.2k 阅读时长 ≈ 1 分钟

关于此攻略

此攻略是由我实际出游太白山然后编写的，同行的都是西安电子科技大学的学生。个人体力不是很好，全程背着三斤多重的单反和两个镜头，尽管很累，但还是坚持尽可能每到一个景点拍照，记录下时间和风景。中间下了好多次雨担心相机淋雨，还有一些难走的石头路，不得不收起相机手脚并用爬下山。总得来说，这篇攻略来之不易，希望能对想去太白山的同学有帮助。

既然作为一个体力不是很好的人（而且我还非常容易晕车），都能从太白山走一趟，所以相信很多想去太白山的同学也没必要太担心，最需要的就是能坚持。

路线示意图

这次太白山出游由我领队，出发前看了网上很多攻略，进行了详细的策划，主要是路线安排，及其出发前要准备的东西。这里先介绍下路线安排，准备物品后面介绍。

路线简介

主要简单介绍几条难度相对小的路线，适合户外经验不多的人员，也是我们选择的路线。详细的介绍请看后面的游记攻略。

  1. 汤峪-下板寺-拔仙台（后面简称“北线”）：这条线应该是难度最小的。

    • 景区外交通：
      西安汽车站（火车站对面）–>汤峪的太白山游客中心：有大巴直达，交通便利。

    • 景区内交通：
      汤峪游客中心–>下板寺（2700m）：有景区中巴；
      下板寺–>索道上站（3200m，离拜仙台不远）：可坐索道；
      拜仙台–>拔仙台（3767.2m）：靠步行，其中有一段正在修建索道。

  2. 拔仙台-南天门-三合宫瀑布-铁甲树-厚畛子-周至（后面简称“南线”）：风景较好，但是相对北线难走、耗时较长，山下交通不如北线方便。下山需要1天，上山需要2天（如果体力较好，提前到达景区，早晨就从铁甲树出发，1天可到）。

    • 景区外交通：
      周至汽车站–>西安水司汽车站：有高速大巴，当然也可以包车。

    • 景区内交通：
      拔仙台–>铁甲树（1600m）：全靠步行。
      铁甲树–>周至：可坐车。

  3. 其他几条线路长，耗时更多。

典型方案

游太白山比较典型的低难度方案有：

  1. 北线原返。

    • 到达天圆地方（3511m）后就原路返回，1天可完成。
    • 登顶后原路返回，一般2天完成（走的快的1天可以完成）。

  2. 北南穿越。从北线上山，从南线下山，一般2天完成。

  3. 南北穿越。从南线上山，从北线下山，一般3天完成。

  4. 南线原返。从南线上山，登顶后原路返回，一般3天完成。

  5. 难度较高方案：南南穿越等其他路线。从一条南线上山，另一条南线下山，通常需要4~7天。

计划路线

计划周六周日两天从西安去游玩太白山。第一天从北线上山；第二天根据情况，选择北线原返或北南穿越，或分两拨行动（安全起见，每拨至少两人互相照应）。

  • 如果天气很不好（刮大风/下雨路滑等），或是第一天走的不够远，考虑到从南线下山时间不够、危险性大，可以先去顶峰看一看，然后统一北线原返。

  • 如果时间足够，但部分同学身体不适，可分两拨下山。

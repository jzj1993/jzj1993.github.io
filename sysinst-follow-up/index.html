装系统不求人——系统安装后续工作

发表于 2015-04-02 分类于 装系统不求人 阅读次数： Valine： 本文字数： 2k 阅读时长 ≈ 2 分钟

装完系统往往还有一些后续工作，比如进入桌面了，显示效果可能不太好，那是因为驱动的原因。

尤其是原版系统，后续工作有点麻烦，但好在都是没有难度的体力活。为了以后常年用电脑顺心，麻烦几天算不得什么~ ｡◕‿◕｡

激活系统

说明：这里的介绍仅供学习使用，请自觉遵守相关法律法规，如果涉及侵权等问题，与本文无关。

这里以Win7为例进行介绍~

查看激活状态

右击桌面或开始菜单中的计算机，选择属性，即可查看系统激活状态。图中是我的电脑，显示Windows已激活。注意这个界面不会自动刷新，如果激活了系统想再次查看激活状态，应该关掉这个窗口，重新打开一个新窗口。

激活系统

  • 如果是你有正版Win7序列号，输入序列号即可激活系统。

  • 如果没有，可以使用附件中的Win7激活工具。

    戳这里可下载附件“Win7激活工具”：
    http://pan.baidu.com/s/1c08NJs0

使用PCSKYS激活系统

  1. 双击打开软件。

  2. 如果你的硬盘活动分区为隐藏分区（即没有盘符），启动本软件会提示发现隐藏分区，是否自动处理，点确定并重启软件即可。

  3. 打开软件后，在OEM厂商标签页选择你的电脑品牌，如果没有你的电脑品牌，那就随便选一个试下吧。

  4. 选好后，设置作用分区为活动分区，点击一键搞定（通常默认就是活动分区，不用设置）。

  5. 一分钟之后，提示激活完成。这时你就可以再次查看系统激活状态了。

    如果仍然显示未激活，重启电脑再看看。

    如果不幸还是显示没激活，用这个工具再试一遍，或者换其他几个激活软件试试。多试几下总是能成功的~

安装驱动

装完系统你可能发现显示效果非常渣，还有热键用不了，声音、亮度调节不了，上不了网等种种奇怪的问题，基本上都是驱动不完善的原因。毕竟新系统本身一般不会出什么故障。

系统自带最基本的驱动，所以装好系统后，你的电脑勉强能用。现在要做的就是给你的电脑装上最合适的驱动，使得所有硬件能充分发挥。

  1. 如果买电脑时附件里面有送驱动光盘，可以直接插上光盘安装驱动。

  2. 如果没有光盘，你可以到电脑品牌的官网下载（当然可以在其他电脑上下载好了复制过来）。

  3. 如果你不会下载和安装驱动，或者嫌麻烦，可以给你的电脑连网后，用驱动人生、驱动精灵等软件自动安装。

  4. 如果你发现电脑上设置网络连接的图标都没有，那可能是网卡驱动的问题。借用别人电脑下载一个万能网卡驱动，拷到你电脑上装好就能连网了。

  5. 不要告诉我你不会连网（让度娘告诉你一切吧）｡◕‿◕｡

  6. 手贱用户请注意，如果你是自己一个个驱动安装的，几乎每个驱动装好后都会默认让你点击确定重启电脑。建议每次确认时注意观察，取消勾选重启电脑的选项，避免反复开关机费时间。所有驱动都装完一并重启才是最好的选择。

官网下载驱动方法简介

  1. 例如我的联想笔记本，首先百度“联想”就可以打开官网了。

  2. 一般会有服务与支持之类的，也有的网站直接是驱动下载什么的，点进去。

  3. 选择驱动软件及手册下载。

  4. 一般会让你选择电脑型号，联想笔记本通常是直接输入主机编号，一般贴在电脑背面的标签上。

  5. 然后会有一堆驱动程序，你可以选择自己的系统（Win7 / 8，32 / 64位等），全部下载下来分别进行安装即可。联想还提供了一键安装驱动的工具，能自动批量安装驱动，会节省不少精力。

安装系统组件、常用软件

所谓系统组件，常见的有NetFramework之类，反正以后再装也不迟。常用软件就不用我说了吧，压缩软件WinRAR、输入法、Office、图片查看器、音乐播放器、视频播放器、QQ、浏览器、杀毒软件等等。

系统设置优化

有关系统设置优化，涉及的东西很繁杂，不过也不是很重要和必要的步骤。例如将桌面等文件夹转移到其他分区，这样重装系统就不用担心保存在桌面的东西丢失了。后面如果有时间，也会写一些相关的文章进行介绍。
